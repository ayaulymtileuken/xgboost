# XGBoost

This project predicts house prices using the XGBoost regression algorithm. The objective was to compare the performance of models trained on log-transformed and raw target values while improving model generalization through hyperparameter tuning, regularization, and early stopping.

Project Workflow
1. Data preprocessing
2. Train/test split with stratification
3. Baseline XGBoost model
4. 5-fold Cross Validation
5. Hyperparameter tuning (GridSearchCV)
6. Regularization
7. Early stopping
8. Model comparison
9. SHAP feature importance analysis
